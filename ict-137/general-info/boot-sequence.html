Boot sequence

Boot sequence

  1. POST - power on self test - stored in ROM
  2. loads the boot loader software
  3. Boot loader initalizes the CPU
  4. Boot loader initializes the FLASH file system
  5. Bootloader loads the OS

POST is layer 1 - anything with the physical components

4 ingreadients of any comp device

  • Processing (CPU)
  • Memory (RAM)
  • Storage (HDD/SSD)
  • Networking (sometimes optional, NIC)

Storage

  • Temp
    • RAM
    • This is volatile - wont survive a shutdown
    • Running config is saved in RAM 
  • Permanent
    • SSD
    • HDD
    • Isn't volatile - will survive a shutdown
    • Know sometimes as ROM - POST is here
    • FLASH is where OS/configs/VLAN database are stored 

OSI model Position/information

  • POST is layer 1 - anything with the physical components
  • LEDs are layer 1
  • Frames layer 2
    • Runts = frame<64 bytes
    • Giant = frame > 1518 bytes
